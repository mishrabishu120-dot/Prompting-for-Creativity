# SkillCraft Technology AI Internship

## Task 02 – Prompting for Creativity

This repository contains my submission for **Task 02 – Prompting for Creativity**, completed as part of the **Prompt Engineering Internship Programme** at **SkillCraft Technology**.

## 📖 Project Overview

The objective of this task was to explore how different prompt engineering techniques influence the creativity, quality, and consistency of responses generated by Large Language Models (LLMs).

The project demonstrates the process of refining prompts from basic instructions to structured, detailed, and few-shot prompts, highlighting how prompt design directly impacts AI-generated content.

---

## 🎯 Objectives

- Understand the importance of prompt structure.
- Compare Basic, Detailed, and Few-shot prompting techniques.
- Generate creative AI startup ideas using structured prompts.
- Analyze the impact of prompt engineering on output quality.
- Learn best practices for designing effective prompts.
